# TechBlog

A Java Servlet and JSP based blogging platform.

## Features
- User Registration & Login
- Add Blog Posts
- Upload Images
- Like and Comment System
- Responsive UI
- Session Management

## Technologies Used
- Java
- Servlet
- JSP
- JDBC
- MySQL
- Bootstrap
- Maven

## Project Structure
- DAO Layer
- Servlet Controllers
- JSP Views
- MySQL Database
